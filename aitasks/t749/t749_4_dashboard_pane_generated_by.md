## Context

Replace the current single-line `Group: explore_001` in the
brainstorm dashboard right pane with a multi-line "Generated by"
block that surfaces the operation type, the agents involved, when it
was run, and a hint that pressing `o` opens the full Operation Detail
screen.

The dashboard pane is the most-used surface for browsing nodes (left
pane = NodeRow list, right pane = the node's metadata). The
"Generated by" block adds context that the user can absorb without
opening any modal.

Depends on t749_1 (br_groups.yaml population) so the lookup works.

## Key Files to Modify

- `.aitask-scripts/brainstorm/brainstorm_app.py` — modify

  `container.mount(Static("Group: ..."))` with a multi-line block.

## Implementation Plan

1. In `brainstorm_app.py`, add a small helper at module scope:

   ```python
   from brainstorm.brainstorm_session import GROUPS_FILE

   def _read_groups(session_path: Path) -> dict:
       p = session_path / GROUPS_FILE
       if not p.is_file():
           return {}
       try:
           data = read_yaml(str(p)) or {}
           return data.get("groups", {}) or {}
       except Exception:
           return {}
   ```

2. In `_show_node_detail` (line 2887), replace the existing
   `Group:` block:
   ```python
   group = node_data.get("created_by_group", "")
   if group:
       container.mount(Static(
           f"[bold $accent]Group:[/] {group}", classes="meta_field"))
   ```
   with:
   ```python
   group = node_data.get("created_by_group", "")
   if group:
       groups = _read_groups(self.session_path)
       ginfo = groups.get(group, {})
       op = ginfo.get("operation", "?")
       agents = ginfo.get("agents") or []
       when = ginfo.get("created_at", "")

       container.mount(Static(
           f"[bold $accent]Generated by:[/] [bold]{op}[/]"
           f"  [dim](group {group})[/]",
           classes="meta_field"))
       if agents:
           container.mount(Static(
               f"[bold $accent]Agents:[/] {', '.join(agents)}",
               classes="meta_field"))
       if when:
           container.mount(Static(
               f"[bold $accent]When:[/] {when}",
               classes="meta_field"))
       container.mount(Static(
           "[dim]Press 'o' for operation details[/]",
           classes="meta_field"))
   ```
   Falls back gracefully when the group entry is missing (legacy
   session): the operation shows as `?`, agents and when are skipped,
   and the `Press 'o'` hint still appears (a still-pressing-o would
   show "no group recorded" message — handled in t749_5).

3. Optional UX touch: color the operation name using the same op-color
   map introduced in t749_3 (`OP_BADGE_STYLES`). Import:
   ```python
   from brainstorm.brainstorm_dag_display import OP_BADGE_STYLES
   ```
   and use the matching color via Rich markup:
   ```python
   op_color = OP_BADGE_STYLES.get(op, UNKNOWN_OP_STYLE).color
   color_hex = op_color.name if op_color else "#888"
   container.mount(Static(
       f"[bold $accent]Generated by:[/] [{color_hex} bold]{op}[/]"
       f"  [dim](group {group})[/]",
       classes="meta_field"))
   ```
